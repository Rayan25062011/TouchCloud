# TouchCloud
A secure cloud storage to add your files and encrypt content

# Use

```python
from cloud import CloudTouch


cloud = CloudTouch()
cloud.cloud()

print("Creating cloud storage...")
print("Adding files...")
cloud.addfile(filename='example.c')

print("files added to cloud")

```
